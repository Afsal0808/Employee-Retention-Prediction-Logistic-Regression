# Employee Retention Prediction using Logistic Regression

## Project Overview

This project analyzes employee retention using Machine Learning.
Exploratory Data Analysis (EDA) is performed to identify factors that influence whether employees leave the company.

A Logistic Regression model is used to predict employee attrition.

---

## Dataset

The dataset contains HR information such as:

* satisfaction_level
* average_montly_hours
* promotion_last_5years
* Department
* salary
* left (target variable)

---

## Project Steps

1. Exploratory Data Analysis (EDA)
2. Salary vs Employee Retention Visualization
3. Department vs Employee Retention Visualization
4. Logistic Regression Model
5. Model Accuracy Evaluation
6. Manual Logistic Prediction using Sigmoid Function

---

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook

---

## Model Accuracy

Accuracy achieved: **~79%**

---

## Logistic Regression Formula

P = 1 / (1 + e^-z)

Where

z = b0 + b1x1 + b2x2 + ... + bnxn
